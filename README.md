## ğŸ”§ Troubleshooting

### Problem: Files not loading (ERR_FILE_NOT_FOUND)
**Solution:** Make sure ALL files are in the same folder. Don't put data files in subfolders.

### Problem: Features not showing
**Solution:** 
1. Check Teacher Mode is enabled
2. Check feature checkboxes are selected
3. Refresh the page (F5 or Cmd+R)

### Problem: No audio playback
**Solution:**
1. Check browser sound is on
2. Try Chrome or Edge (best Web Speech API support)
3. Check browser microphone permissions (for recording)

### Problem: Translation not showing
**Solution:**
1. Teacher Mode â†’ Enable "Home Language Support"
2. Select a language from dropdown
3. Play a word - translation appears in result modal
4. Note: Only 8 core words currently translated (cat, dog, sun, ship, rain, tree, bird, hope)

### Problem: Syntax Error in console
**Solution:** 
- This package has the syntax error fixed
- If you see errors, ensure you're using the latest files
- Clear browser cache (Ctrl+Shift+R / Cmd+Shift+R)
